# -----------------------------------------------------------------------------
# 14. Setup and Run
# -----------------------------------------------------------------------------
# 1. Create a new Django project: django-admin startproject crm_project
# 2. Create the apps: python manage.py startapp users, teams, customers, core
# 3. Add the apps to INSTALLED_APPS in crm_project/settings.py
# 4. Set AUTH_USER_MODEL = 'users.User' in settings.py
# 5. Copy the code into the respective files.
# 6. Create templates directory and subdirectories.
# 7. Run migrations: python manage.py makemigrations and python manage.py migrate
# 8. Create a superuser: python manage.py createsuperuser
# 9. Run the server: python manage.py runserver
# 10. Use the admin panel to create users with different roles, teams, and groups.
